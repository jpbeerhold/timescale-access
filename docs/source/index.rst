Welcome to timescale-access's documentation!
===========================================

`timescale-access` is a small Python library providing a convenience wrapper
around TimescaleDB/PostgreSQL using SQLAlchemy. It is designed for working
with time-series data, especially in quant/trading and analytics pipelines.

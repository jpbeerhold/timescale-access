## [0.1.0] - 2025-11-29
### Added
- Initial release-ready project structure.
- Core `TimescaleAccess` client for interacting with TimescaleDB/PostgreSQL.
- Automatic hypertable creation and schema management helpers.
- High-performance DataFrame inserts and conflict-safe upserts.
- Advanced SQL analysis utilities (missing sequences, duplicates, null summary, hypertable size).
- Complete set of integration tests using a TimescaleDB test container.
- Full Sphinx documentation with autodoc and Read the Docs theme.
- GitHub Pages deployment workflow for publishing documentation.
- GitHub Actions CI pipeline with TimescaleDB service for automated testing.
- Ruff linting workflow for static analysis.
- VS Code Devcontainer setup with TimescaleDB and automatic dependency installation.
- GHCR-ready runtime Dockerfile.
- MIT license, README, and project metadata.
